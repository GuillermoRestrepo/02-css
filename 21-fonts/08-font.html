PROPIEDAD - (font)

(font) es una propiedad abreviada (shorthand), la cual permite definir en una sola línea, varias propiedades que estilizan el texto de un elemento.

¿Qué Propiedades Combina La Propiedad font?

La propiedad font combina las siguientes propiedades individuales (en este orden):

  1. font-style (opcional)
  2. font-weight (opcional)
  3. font-size (obligatorio)
  4. line-height (opcional)
  5. font-family (obligatorio)

        font: [font-style] [font-weight] [font-size]/[line-height] [font-family;

Ejemplo Mínimo Requerido

Como mínimo, se debe de especificar el tamaño de la letra (font-size) y la fuente de esta (font-family), lo demás es opcional:


        p {
            font: 14px Arial, sans-serif;
        }

Al igual que en en la propeidad font-family, es recomendable especificar una fuente de respaldo (fallback) separada por una coma.

Ejemplo Completo


        p {
            font: italic bold 16px/1.5 "Georgia", serif;
        }

Lo cual es equivalente a:


        p {
            font-style: italic;
            font-weight: bold;
            font-size: 16px;
            line-height: 1.5;
            font-family: "Georgia", serif;
        }

NOTA

  • No es aconsejable utilizar la propiedad font, ya que por descuido, se puede afectar a elementos hijos o se pueden reescribir los estilos de otros elementos.
  • Como se puede notar en los ejemplos, la propidades font-size y lineheigh, son separadas por un slash (/).
