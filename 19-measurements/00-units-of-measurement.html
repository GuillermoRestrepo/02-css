UNIDADES DE MEDIDA

Las unidades de medida en CSS se usan para definir longitudes: ancho (width), alto (height), márgenes, paddings, fuentes, entre otras.

Estas se dividen en dos grandes categorías:

Medidas Absolutas

Sirven para indicar medidas fijas, que no cambiarán al momento de ser visualizadas en diferentes dispositivo o pantallas de diferente tamaño.

Unidad Descripción Equivalencia Aproximada
px Píxeles (unidad más usada) 1px = 1 punto en pantalla
cm Centímetros 1cm = 37.8px
mm Milímetros 1mm = 3.78px
Q Cuarto de milimetro 1Q = 1/40 de 1cm
in Pulgadas 1in = 2.54cm = 96px
pt Puntos tipográficos 1pt = 1/72in ≈ 1.33px
pc Picas 1pc = 12pt ≈ 16px

NOTA: al utilizar medidas absolutas, NO es aconseble el uso de otras unidades que no sean píxeles (px), ya que, no se adaptan correctamente a diseños resposivos.

Medidas Relativas

Sirven para indicar medidas que se adaptan al tamaño del elemento que las contiene, excelentes para responsividad.

Unidad Relativa a…
% El contenedor padre
em El tamaño de fuente del elemento actual
rem El tamaño de fuente raíz (html)
vw 1% del ancho del viewport
vh 1% del alto del viewport
vmin 1% del valor menor entre vw y vh
vmax 1% del valor mayor entre vw y vh
ch Ancho del carácter 0 del font actual
lh Altura de una línea del elemento
ex Altura de la x minúscula (menos común)

Las medidas relativas mas útilizadas son: %, em, rem, vw, vh, vmin y vmax.
