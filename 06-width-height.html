PROPIEDADES width & height

Estas propiedades sirven para establecer el tamaño de un elemento.

  • width: Define el ancho de un elemento.
  • height: Define el alto de un elemento.

¿Cómo funcionan?

Por defecto, width y height se aplican al contenido del elemento. A menos que se utilice la propiedad box-sizing con el valor border-box.

IMPORTANTE

  • Las propiedades (width) y (height) no funcionan en elementos de línea, ya que, los elementos de línea se caracterizan por adaptarse al tamaño de su contenido. Ejemplo:

    Código HTML:

    
<body>
    <div class="bloque">ESTE ES UN ELEMENTO DE BLOQUE</div>
    <span class="linea">ESTE ES UN ELEMENTO DE LÍNEA</span>
</body>

    Código CSS:

    
.bloque, .linea{
    width= "200px";
    height= "200px";
}
    ESTE ES UN ELEMENTO DE BLOQUE
    ESTE ES UN ELEMENTO DE LÍNEA

    Como se puede notar, el elemento de bloque si se ve aectado por las propiedades (width) y (height), mientras que el elemento de línea no.

  • En caso que se desee dar medidas a un elemento de línea, este tiene que contar con la propiedad display con el valor block o inline-block. (VEREMOS EL USO DE ESTO MÁS ADELANTE)

Valores Para Las Propiedades width / height

  • Pixeles (px): tamaño fijo.
  • Porcentaje (%): toma un tamaño porcentual del elemento q lo contiene.
  • Viewport units (vw, vh):
    • 1vw = 1% del ancho de la pantalla.
    • 1vh = 1% del alto de la pantalla.

  • Palabras Clave

    • auto: (por defecto) el navegador calcula el tamaño.
    • inherit: hereda el tamaño del elemento padre.
    • initial: restablece al valor por defecto del navegador.
    • unset: elimina cualquier valor, respetando herencia.
