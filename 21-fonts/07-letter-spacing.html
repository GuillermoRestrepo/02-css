PROPIEDAD - (letter-spacing)

Sirve para aumentar o dismiuir el espacio horizontal que existe entre cada letra / caracter / glafo de un texto.

Valores Posibles:

Esta propiedad acepta medidas, ya sean absolutas o relativas, sin embargo se suele utilizar px en caso de que se desee una medida absoluta y em o rem en caso de que se desee una medida relativa.

Ejemplo:


letter-spacing: -2px;
                

letter-spacing: 2px;
                

letter-spacing: -0.2em;
                

letter-spacing: 0.2em;
