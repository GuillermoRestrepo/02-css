COLORES CON (hsl)

HSL significa:

  • H = Hue (Tono)
  • S = Saturation (Saturación)
  • L = Lightness (Luminosidad)

Es un modelo de color más intuitivo que RGB, ya que permite manipular el tono, la viveza y la claridad del color de forma separada.

Sintaxis

  • hsl(h s l)
  • hsl(h, s, l)
  • hsla(h s l / transparencia)
  • hsla(h, s, l, transparencia)

Tono

Representa el color básico en un círculo cromático de 360°:

Hue (°) Color
0° Rojo
60° Amarillo
120° Verde
180° Cian
240° Azul
300° Magenta
360°/0° Rojo otra vez

Saturación

Determina qué tan intenso o apagado es el color:

  • 100% = color totalmente saturado
  • 0% = gris (sin color)

Luminosidad

Determina qué tan claro u oscuro es el color:

  • 0% = negro
  • 50% = color normal
  • 100% = blanco

Ejemplos Y Sintaxis

SIN Opacidad / Transparencia

Red


hsl(0 100% 50%)

Yellow


hsl(60, 100%, 50%)

Green


hsl(120 100% 50%)

Cyan


hsl(180, 100%, 50%)

Blue


hsl(240 100% 50%)

Magenta


hsl(300, 100%, 50%)

CON Opacidad / Transparencia

Rojo con 50% transparencia


hsl(0, 100%, 50%, 0.5)

Verde con 50% transparencia


hsl(120 100% 50% / 0.5)

Azul con 50% transparencia


hsla(240, 100%, 50%, 0.5)
