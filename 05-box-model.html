Box-Model

El Box Model (Modelo de Caja) es la forma en que CSS ve y calcula el tamaño de todos los elementos en una página web.

Cada elemento se considera como una caja compuesta por 4 partes:

  1. Contenido (content):: El área donde está el texto, imagen, etc.
  2. Relleno (padding):: Espacio entre el contenido y el borde.
  3. Borde (border):: El borde que rodea al padding y al contenido.
  4. Margen (margin):: Espacio exterior que separa un elemento de otros.

¿Cómo se calcula el tamaño total de un elemento?


        Tamaño total = width + padding izquierdo + padding derecho + border izquierdo + border derecho
