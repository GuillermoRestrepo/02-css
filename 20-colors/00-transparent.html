transparent

transparent no es un color como tal, simplemente vuelve invisible/transparente el elemento o la parte del elemento q cuente con este.

Ejemplos


.triangulo-1 {
    border-top: 125px solid red;
    border-bottom: 125px solid red;
    border-left: 125px solid blue;
    border-right: 125px solid blue;
}
                

.triangulo-2 {
    border-bottom:250px solid red;
    border-left: 125px solid;
    border-right: 125px solid;
    border-left-color: transparent;
    border-right-color: transparent;
}
                

.triangulo-3 {
    border-top:250px solid red;
    border-left: 125px solid;
    border-right: 125px solid;
    border-left-color: transparent;
    border-right-color: transparent;
}
                

.triangulo-4 {
    border-left: 250px solid red;
    border-top: 125px solid;
    border-bottom: 125px solid;
    border-top-color: transparent;
    border-bottom-color: transparent;
}
                

.triangulo-5 {
    border-right: 250px solid red;
    border-top: 125px solid;
    border-bottom: 125px solid;
    border-top-color: transparent;
    border-bottom-color: transparent;
}
