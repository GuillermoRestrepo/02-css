height: %;

Hace que el elemento ocupe un porcentaje del tamaño del elemento q lo contiene.

height: auto;

Hace que el elemento ocupe el tamaño de su contenido.

NOTAS

  • Utilizar (%) solo funciona si el elemento contenedor tiene un tamaño definido.

  • Si el tamaño del elemento contenedor no está definido, cuaquier medida porcentual que se le de al elemento hijo, será tomada como (auto).

  • En la mayoria de ocaciones no es necesario ocupar la proiedad (height) ya q por defecto su valor es (auto) y ocupará el espacio que requiera su contenido.

  • En caso de que se desee espacios más allá del que ocupa el contenido, es preferible utilizar la propiedad (padding).

  • IMPORTANTE: Al utilizar (position: absolute;) en el elemento hijo, el porcentaje se calculará respecto al tamaño del viewport y no respecto al contenedor.

Ejemplos:


.box-1 {  
    width: 50px;
    height: 50px;
}

.box-2 {
    width: 50%;
    height: 50%;
}
                

.box-3, .box-4 {
    width: 100%;
    height: auto;
}
                

.box-5 {
    width: 50px;
    height: 100%;
    position: absolute;
    top: 0px;
    left: 0px;
}

.box-6 {
    width: 50px;
    height: 100%;
}
