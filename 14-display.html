Propiedad (display)

Permite que los elementos de línea cambien a elementos de bloque y viceversa.

NOTA: Tabién sirve para posicionar los elementos con flex y grid, pero eso se verá mas adelante.

Posibles Valores

Nota:Recordemos que los elementos de línea no pueden tener medidas, ni margenes verticales.

display: block;

Convierte un elemento de línea en un elemento de bloque. Ejemplo:

Código SIN (display: block;) CON (display: block;)

Código En HTML:


<body>
    <span class="el1">Elemento De Línea #1</span>
    <span class="el1">Elemento De Línea #2</span>    
</body>

Código En CSS:


.el1 {
    background-color: orange;
    width: 150px;
    height: 150px;
    margin: 10px;
}
Elemento De Línea #1 Elemento De Línea #2 Elemento De Línea #1 Elemento De Línea #2

display: inline;

Convierte un elemento de bloque en un elemento de línea. Ejemplo:

Código SIN (display: inline;) CON (display: inline;)

Código En HTML:


<body>
    <div class="eb1">Elemento De Bloque #2</div>
</body>

Código En CSS:


.eb1 {
    background-color: orange;
    width: 150px;
    height: 150px;
    margin: 10px;
}

display: inline-block;

Hace que cualquier elemento, ya sea de línea o de bloque, se comporte como un elemento de línea, pero que pueda tener medidas y margenes como si fuera un elemento de bloque. Ejemplo:

Código SIN (display: inline-block;) CON (display: inline-block;)

Código En HTML:


<body>
    <span class="el3">Elemento De Línea</span>        
    <div class="eb3">Elemento De Línea</div>
    
</body>

Código En CSS:


.el1 {
    background-color: orange;
    width: 150px;
    height: 150px;
    margin: 10px;
}

display: none;

Hace q el elemento no sea visible, se sigue cargando, pero no es visible ni interactuable para el usuario (Útil para activar por medio de Java Script). Ejemplo:

Código SIN (display: none;) CON (display: none;)

Código En HTML:


<body>
    <span class="el3">Elemento De Línea</span>        
    <div class="eb3">Elemento De Línea</div>
    
</body>

Código En CSS:


.el1 {
    background-color: orange;
    width: 150px;
    height: 150px;
    margin: 10px;
}
