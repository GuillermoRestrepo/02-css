Propiedad padding

Sirve para generar espacio interior entre el contenido de un elemento y su borde.

NOTA: A diferencia de margin la propiedad padding si se aplica a los cuatro lados de un elemento de línea

NOTA 2: A diferencia de margin la propiedad padding si modifica el tamaño del elemento.

Funcionamiento

  • Un valor:

    
                /*
                    aplica a todos los lados
                */
                padding: 40px;

  • Dos valores:

    
                /*
                    1er valor: padding superior e inferior.
                    2do valor: padding izquierdo y derecho.
                */
                padding: 40px 30px;

  • Tres valores:

    
                /*
                    1er valor: padding superior.
                    2do valor: padding izquierdo y derecho.
                    3er valor: padding inferior
                */
                padding: 40px 30px 20px;

  • Cuatro valores:

    
                /*
                    1er valor: padding superior.
                    2do valor: padding derecho.
                    3er valor: padding inferior.
                    4to valor: padding izquierdo.
                */
                padding: 40px 30px 20px 10px;

Propiedades Individuales


        /*Margen superior*/
        padding-top: 10px;

        /*Margen derecho*/
        padding-right: 15px;

        /*Margen inferior*/
        padding-bottom: 20px;

        /*Margen izquierdo*/
        padding-left: 25px;

Valores Posibles

  • px: valores fijos pixeles.
  • %: porcentaje del ancho del contenedor.

NOTA: la propiedad padding no admite el valor auto.

Malas Practicas

Darle padding 0 al selector universal.

Evitar reescribir paddings verticales u orizontales al usar diferentes cantidades de valores, 2 3 o 4 valores.

Buena Practica

Es recomendable utilizar padding:0; en el elemento body, en caso de q este tenga un valor por defecto q disminya la pantalla pintable.
