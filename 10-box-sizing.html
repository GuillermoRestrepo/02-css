PROPIEDAD (box-sizing)

Esta propiedad define cómo se calcula el tamaño total de un elemento.

Posibles Valores

  • content-box (Valor Por Defecto)

    Hace que las propiedades (width) y (height) solo se apliquen al contenido del elemento, es decir, para calcular el valor total del elemeto, habría q sumar el tamaño del contenido (width / height), el margen interno (padding) y el borde (borde) de este mismo. Ejemplo:

    Código HTML:

    
<body>
    <div class="eb1">Bloque #1</div>
</body>

    Código CSS:

    
.eb1 {
    background-color: orange;
    width: 150px;
    height: 150px;
    padding: 20px;
    border: 10px solid black;
}
    Bloque #1

    En este ejemplo se puede ver que el tamaño real del elemento es de 210x210 píxeles a pesar de haber asignado 150 píxeles en las propiedades (width & height)

  • border-box

    Este valor hace que las propiedades (width) y (height) definan el tamaño total del elemento.

    Código HTML:

    
<body>
    <div class="eb2">Bloque #2</div>
</body>

    Código CSS:

    
.eb2 {
    box-sizing: border-box;
    background-color: orange;
    width: 150px;
    height: 150px;
    padding: 20px;
    border: 10px solid black;
}
    Bloque #2

    En este ejemplo se puede notar como el elemento mide exactamente lo que se asignó a las propiedades (width & height), aun contando con el mismo borde y el mismo margen interno del primer ejemplo.
