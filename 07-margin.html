margin

Generar espacio alrededor de un elemento. Es decir, sirve para separar un elemento de otro.

Nota: a los elementos de línea unicamente se le puede aplicar margenes horizontales (Izquierdo y derecho).

¿Como Funciona?

  • Si la propiedad margin tiene un solo valor:

    
                /*
                    Aplica a todos los lados, (arriba, abajo, izquierda, derecha)
                */
                margin: 20px;

  • Si la propiedad margin tiene dos valores:

    
                /*
                    El primer valor: El margen superior e inferior.
                    El segundo valor: El margen izquierdo y derecho.
                */
                margin: 20px 10px;

  • Si la propiedad margin tiene tres valores:

    
                /*
                    El primer valor: El margen superior.
                    El segundo valor: El margen izquierdo y derecho.
                    El tercer valor: El margen inferior.
                */
                margin: 20px 10px 30px;

  • Si la propiedad margin tiene cuatro valores:

    
                /*
                    El primer valor: El margen superior.
                    El segundo valor: El margen derecho.
                    El tercer valor: El margen inferior.
                    El cuarto valor: El margen izquierdo.
                */
                margin: 20px 10px 30px 40px;

Propiedades Individuales

Util cuando se necesita generar un margen a uno o varaios lados especificos de un elemento.


        /*Margen superior:*/
        margin-top: 10px;

        /*Margen derecho:*/
        margin-right: 15px;

        /*Margen inferior:*/
        margin-bottom: 20px;

        /*Margen izquierdo:*/
        margin-left: 25px;

Posibles valores

  • px: medida fija en pixeles.
  • %: pocentaje relativo al tamaño del contenedor.

  • auto: hace q el navegador calcule el margen del elemento.

    NOTAS SOBRE VALOR AUTO:

    • El valor (auto) no funciona para los elementos de línea.
    • El valor (auto) solo funciona si el elemento tiene tamaño horizontal definido (width).
    • El valor (auto) no funciona para los lados verticales (top / bottom).
    • Se suele utilizar el valor (auto), para centrar un elemento o para apegarlo a un lado en especifico.

Malas Pácticas

  • Al selector universal (*) no se le da (margin: 0px;), porq le quitará el margen a todos los elementos y no se sabría cuando empieza uno y termina el otro.

  • No se utiliza (margin: 0px auto;), porq reescribe los margenes verticales.

Buena Práctica

Darle al elemento <body> un margin: 0px; si es recomendable, ya que, este debe de ocupar toda la pantalla displonible para pintar contenido.
