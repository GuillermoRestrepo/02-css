Valor currentColor

Al poner este valor en alguna propiedad que requiera un color, hará que el color de esta propiedad sea del mismo color que la letra del elemento ancestro mas cercano que tenga la propiedad color definida.

Ejemplo

En este ejemplo se puede ver como el cuadradito del centro que es el elemento hijo, tomará el color de la propiedad color del primer ancestro que tenga un color definido.

Si ningun ancestro tiene la propiedad color definida, este tomará el color por defecto del navegador que en la mayoria de navegadores es negro.


.hijo {
    background-color: currentColor;
    width: 25px;
    height: 25px;
}

.bisabuelo-1 {
    color: red;
}

.abuelo-1 {
    color: green;
}

.padre-1 {
    color: blue;
}
