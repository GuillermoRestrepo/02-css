PROPIEDAD - (font-size)

Esta propiedad se utiliza para definir el tamaño de un texto.

Unidades De Medida

Absolutas Relativas
px em
pt rem
cm %
mm vw
in vh

Valores Textuales

Valor Relación aproximada con el texto normal
xx-small Muy, muy pequeño
x-small Muy pequeño
small Pequeño
medium Normal (por defecto)
large Grande
x-large Muy grande
xx-large Muy, muy grande
larger Más grande que el padre
smaller Más pequeño que el padre

Ejemplo


font-size: xx-small;
                

font-size: x-small;
                

font-size: small;
                

font-size: medium;
                

font-size: large;
                

font-size: x-large;
                

font-size: xx-larg;

Notas:

  • Evitar el uso de px, ya que no se escala bien en diferentes tamaños de pantalla.
  • Usar rem o em para lograr un diseño más fluido y adaptativo.
  • Mantener el tamaño de la fuente base (html { font-size: 100%;}) para ser tomado como referencia.
