SELECTORES

Como ya lo sabemos, los selectores son la forma en la que CSS identifica los elementos de un documento HTML, para que estos puedan ser modificados, no obstante, existen diferentes tipos de selectores, los cuales veremos a continuación:

Selector Universal

Se representa con un asterisco (*), y afecta a todos los elementos del documento. Ejemplo:


                    * {
                        background-color: yellow;
                    }

Esta regla hará que el fonde sea de color amarillo.

Selector De Etiqueta

Se representa con el nombre de la etiqueta, y afecta unicamenta a las etiquetas que lleven el mismo nombre. Ejemplo:


                    <p> {
                        background-color: blue;
                    }

Esta regla hará que todo el contenido de las etiquetas <p> tenga un fondo de color azul.

Selector De ID

Se representa con un numeral/almoadilla (#), seguido del valor del atributo (id="valor"), recordemos que el valor de este atributo debe de ser unico para cada elemento. Ejemplo:

En HTML:


                    <body>
                        <h2 id="encabezado-1">Encabezado 1</h2>
                        <h2 id="encabezado-2">Encabezado 2</h2>
                    </body>

En CSS:


                    #encabezado-1 {
                        color: green;
                    }
                    #encabezado-2 {
                        color: orange;
                    }

Estas reglas harán, que las letras del encabezado 1 sean de color verde y las letras del encabezado 2 sean de color anaranjado.

Selector De Clase

Se representa con un punto (.), seguido del valor del atributo (class="valor"), este si puede ser utilizado en más de un elementos. Ejemplo:

En HTML:


                    <body>
                        <h2 class="encabezado">ENCABEZADO 1</h2>
                        <h2 class="encabezado">Encabezado 2</h2>
                    </body>

En CSS:


                    .encabezado {
                        color: purple;
                    }

Esta regla hará que ambos encabezados sean de color morado.

Selector Por Medio De Cualquier Atributo

Se caracteriza por estar encerrado entre corchetes, y existen 6 formas de utilizar este, iremos una por una:

Forma Sintaxis Como funciona
1 [href] Afecta a todos los elementos que cuenten con el atributo (href)
2 [href="encabezado"] Afecta a todos los elementos que cuenten con el atributo (href) y a su vez, que el valor de este sea el que se especifica en el selector. En este caso el valor especificado es: (encabezado).
3 [href^="enca"] Afecta a todos los elementos que cuenten con el atributo (href) y a su vez, que el valor de este comience con el que se especifica en el selector. En este caso el valor especificado es: (enca).
4 [href$="zado"] Afecta a todos los elementos que cuenten con el atributo (href) y a su vez, que el valor de este termine con el que se especifica en el selector. En este caso el valor especificado es: (zado).
5 [href*="be"] Afecta a todos los elementos que cuenten con el atributo (href) y a su vez, que el valor de este contenga en alguna parte el valor que se especifica en el selector. En este caso el valor especificado es: (be).
6 [href|="encabezado"] Afecta a todos los elementos que cuenten con el atributo (href) y a su vez, que el valor de este comience con el que se especifica en el selector, seguido de un guión (-) y texto adicional. En este caso el valor especificado es: (encabezado), pero podría afectar a (encabezado-cualquier cosa).

Selectores Agrupados

Se trata de dos o más selectores seprados por una coma (,), se utiliza cuando varios elementos tienen estilos diferentes pero comparten un estilo entre si. Ejemplo:

En HTML:


                    <body>
                        <h2 class="encabezado-1">ENCABEZADO 1</h2>
                        <h2 class="encabezado-2">Encabezado 2</h2>
                    </body>

En CSS:


                    .encabezado-1, .encabezado-2 {
                        background-color: black;
                    }

                    .encabezado-1 {
                        color: green;
                    }
                    
                    .encabezado-2 {
                        color: yellow;
                    }

Estas reglas harán que ambos encabezados tengan un fondo de color negro, pero el color de la letra del 1er encabezado será verde y del sgundo será amarilla.

Nota: Pueden combinarce diferentes tipos de selectores. Ejemplo: (h1, .clase, #identificador, [src])

Selector Descendente

Sirve para especificar elementos que se encuentren dentro de otros elementos, esto se hace separando los selectores unicamente por un espacio. Ejemplo:

En HTML:


                    <body>
                        <h2>Encabezado 1</h2>
                        <div class="container">
                            <h2>Encabezado 2</h2>
                        </div>
                    </body>

En CSS:


                    .container h2 {
                        background-color: red;
                    }

Esta regla hará que el fondo de la etiqueta <h2> sea de color rojo, pero solo la etiqueta <h2> que se encuentre dentro de un <div> y como se puede ver, tambien se pueden combinar los selectores.

Nota: las buenas practicas indican q como maximo se debe de bajar un unico nivel, descender mas niveles puede generar problemas.

Selector Hijos Directos

Afecta a los hijos directos, esto se hace poniendo el selector del elemento padre, seguido de un mayor que (>), seguido del selector del elemento hijo. Ejemplo:

En HTML:


                    <body>
                        <div class="container">
                            <h1>Encabezado 1</h1>
                            <p>Parrafo #1</p>

                            <div>
                                <h2>Encabezado 2</h2>
                                <p>Parrafo #2</p>

                            </div>
                            <p>Parrafo #3</p>
                        </div>
                    </body>

En CSS:


                    .container > p {
                        color: black;
                    }

A pesar de q los 3 parrafos están contenidos por una etiqueta (<p>), unicamente los estilos se aplicarán al primer y al tercer parrafo, ya que, estos sí son hijos directos del elemento <div> que tiene la clase (.container), mientras que el parrafo 2, vendria a ser un nieto.

Nota: tambien se pueden combinar selectores y tambien se puede descender mas niveles (nietos, bisnietos y demas), pero igual es aconsejable solo descender un nivel.

Selector De Hermanos

Afecta unicamente a los elementos hermanos, es decir, los elementos q compartan el mismo elemento contenedor. hay dos formas.

  1. Hermano Siguiente

    Selecciona el elemento hermano siguiente mas próximo, que se haya indicado.

    
                            p + h2 {
                                color: blue;
                            }

    Le dará estilos al próximo elemento hermano que sea un <h2>

  2. Hermanos Siguientes

    Selecciona los elementos hermanos siguientes, que se haya indicado.

    
                            .encabezado-2 ~ p {
                                color: red;
                            }

    Le dará estilos a los siguientes elementos <p> q sean hermanos del encabezado <h2> que tenga la clase encabezado-2
