NOTA: la opacidad / trasparentar puede servir para darle efectos a botones, sombras de elementos, relieves y demas.

GAMA DE COLORES (R G B)

Se basa en 3 segmentos (Red, Green, Blue), de 8 bits cada segmento, cada bit puede tener el valor de 0 o 1, yendo desde (00000000) hasta (11111111).

Representados en el sistema decimal de 0 a 255.

Y existe un 4to valor que sirve para representar la trasparencia del elemento, este puede ir de 0 a 1.

Nomenclatura

  • rgb(R G B)
  • rgb(R, G, B)
  • rgb(R, G, B, A)
  • rgba(R, G, B, A)
  • rgb(R G B / A)

Ejemplos


.box-1 {
    background-color: rgb(0 0 0);
}
                

.box-2 {
    background-color: rgb(255, 255, 255);
}
                

.box-3 {
    background-color: rgb(255, 0, 0, 0.25);
}
                

.box-4 {
    background-color: rgba(0, 255, 0, 0.5);
}
                

.box-5 {
    background-color: rgb(0 0 255 / 0.75);
}
