MENÚ DE NAVEGACIÓN

position: fixed;

  • Pierde su espacio reservado.
  • Permite el uso de (top), (right), etc.
  • Los elementos posicionados con (fixed), toman como referencia el viewport del navegador (es espacio pintable), y se mantiene de forma flotante por mas q se scrolee.
  • Los valores de (top), (top), etc, se calculan a partir de los bordes del contenedor del elemento, es decir, del vewport.

Ejemplo

Los ejemplos son la barra de navegacion y los cuadrados en la esquina q aparecen que no se moveran cuando se hace scroll.

Código

  • Código de "Menú De Navegación"

    
.encabezado {
    background-color: #333;
    color: #fff;
    width: 100%;
    height: 40px;
    padding: 10px;
    position: fixed;
    text-align: center;
}

  • Código de "Menú De Cuadrados"

    
.cuadrado1, .cuadrado2 {
    width: 200px;
    height: 200px;
    position: fixed;
}

.cuadrado1 {
    background-color: lightsalmon;
    bottom: 20px;
    left: 20px;
}

.cuadrado2 {
    background-color: lightseagreen;
    bottom: 20px;
    right: 20px;
}

Elemento De Línea Usando (position: fixed;)

Admite medidas como cuando se lo usa con (position: absolute;) solo q ahora no se desplaza y queda referemte al viewport.
