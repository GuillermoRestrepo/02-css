position: absolute;

  • Pierde su flujo de renderizado.
  • Pierde su espacio reservado.
  • Permite el uso de top, left, etc.
  • Al utilizar las propiedades top, left, etc, los elementos se moveran tomando como referencia los bordes de su contenedor.
  • Los elementos posicionados con (absolute), toman como contenedor al primer elemento antecesor que se encuentre posicionado con (relative, absolute o fixed).
  • En caso de q ningun elemento antecesor se encuentre posicionado, este tomará como contenedor referente el elemento html, es decir (toda la pantalla dibujable).

Ejemplos:

  • El bloque gris oscuro representa el viwport del navegador (toda la pantalla dibujable).
  • El primer contenedor gris claro, se encuentra posicionado con (possition: relative;).
  • El segundo contenedor gris claro, no se encuentra posicionado, lo q es igual a (position static).
  • El bloque 1 y 2 de color salmon y turquesa respectivamente, se encuentran centrados y sin posicinarse.

Elemento De Línea Con (position: absolute;)

Hace q un elemento de línea se comporte como un elemento de bloque.

.l1 {

    height: 30px;
    text-align: center;
}

    position: absolute;
