CONTEXTO DE POSICIONAMIENTO

Se refiere a como actua la propiedad (z-index), en diferentes contextos, contexto los cuales están definidos por las propiedades que usa un elemento, o por el contenedor en el q se encuentra.

Ejemplo rápido

Código en HTML:


<body>
    <div class="contenedor-1">
        <div class="caja-1"> </div>
    </div>
    <div class="contenedor-2"> </div>
</body>

Código en CSS:


.contenedor-1, .contenedor-2, .caja-1 {
    position: relative;
}

.contenedor-1 {
    z-index: 1;
}

.contenedor-2 {
    z-index: 2;
}

.caja-1 {
    z-index: 99999;
}

No importa que tan grande sea el valor de la propiedad (z-index) del elemento (caja-1) jamas podra posicionar su contenido por delante del conteido del elemento (contenedor-2) porq son contextos diferentes en los q se aplica dicha propiedad.

Otras Propiedades Que Pueden Crear Diferentes Contextos De Apilamiento

  • opacity menor que 1
  • transform (como transform: scale(1))
  • filter
  • perspective
  • mix-blend-mode distinto de normal
  • will-change con ciertos valores
  • contain: layout, paint, etc.

NOTA IMPORTANTE

Hay q tener muy en cuenta cada contexto de apilamiento que se crea, esto evitará conflictos al querer posicionar un elemento delante de otro.
