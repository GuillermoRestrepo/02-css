PROPIEDAD (text-align)

Sirve para alinear el contenido de un elemento.

Posibles Valores

Alineado A La Izquierda (Valor Por Defecto)

text-align: left

Centrado

text-align: center

Alineado A La Derecha

text-align: right

Justificado

text-align: justify

Notas

  1. Cuando No Se Usa (text-align)
    • No se usa en elementos de linea.
    • No se usa en elementos que tengan la propiedad (display) con el valor (inline o inline-block)
  2. Posibles Soluciones:
    • Si no es un elemento de bloque, hay que hacer que sea de bloque con (display: block;).
    • Encapsular el elemento de línea en un elemento de bloque y aplicar la propiedad (text-align) en el elemento de bloque. (SOLUCION NO RECOMENDADA)
  3. No Hay Que Confundir Alinear Contenido Con Alinear Contenedor:
    • Para centrar un contenido se utiliza la propiedad (text-align: center;)
    • Para centrar un contenedor se usa la propiedad (margin-(left/right): auto;).

Adicional

Las imágenes y vídeos tienen por defecto la propiedad (display: inline-block;), en caso de que se desee aplicar la propiedad (text-align), habrían convertirlos en elementos de bloque (display: block;)
