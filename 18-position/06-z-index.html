PROPIEDAD (z-index)

  • Sirve para indicar que elemento se mostrará delante de otro sobreponiendose a este.
  • Solo funciona en elementos que se encuentren posicionados.
  • El valor más alto, será el que se posicone al frente.
  • Es importante dejar una diferencia considerable entre el valor (z-index) de un elemento y de otro.
  • Este espacio puede ir de 10 en 10 o preferible de 100 en 100.
  • Esto se hace para no tener la necesidad de cambiar cada valor de (z-index), en caso de que posteriormente, se necesite añadir otro elemento.
  • Si se coloca un valor negativo, el elemento se posicionará por detras de los elementos no posicionados, e incluso detras de su contenedor.

Ejemplos

Box-1 y Box-2 no se encuentran posicionados, pero Box-2 tiene un (margin-top: -20px;) haciendo que ocupe el espacio de Box-1 y sobreponiendose a este, ya que, con elementos no posicionados, siempre se sobrepone el último elemento colocado en el HTML, en este caso Box-2. Box-3 se encuentra posicionado, pero Box-4 no, esto quiere decir que un eleento posicionado se sobrepone a un elemento sin posicionar. Box-5 y Box-6 están posicionados, nuevamente se posicionará por delante el último elemento colocado en el html, en este caso Box-6. Box-7 y Box-8 están posicionados, pero Box-7 tiene la propiedad (z-index) con el valor de 1, esto significa que este se sobrepondra a Box-4. Box-9 y Box-10 están posicionados, pero Box-9 tiene la propiedad (z-index) con el valor de 1 y Box-10 tiene con el valor de 2, demostrando q entre mayor sea el valor de esta propiedad, mayor prioridad se tendra para mostrarse por delante.

Ejemplo Con Elementos Padre E Hijo

Como se ve normalmente, siendo Caja-1 y Caja-2 elementos hermanos que a su vez son elementos hijos del contenedor. El contenedor y las cajas estan posicionados, para hacer que las cajas se desborde el elemento padre no tiene la propiedad (overflow) o si la tiene que la tenga con el valor (visible). Caja-5 y Caja-6 tiene un valor de (-1), en la propiedad (z-index), es por eso que se colocan detras del elemento padre, para q esto funcione, el elemento padre no debe de tener la propiedad (z-index) o tenerla con el valor de (auto) que es el valor por defecto. Caja-7 tiene un valor de (-1) y Caja-8 tiene un valor de (-2), por eso Caja-7 se sobrepone a Caja-8, pero ambas siguen estando detras del elemento padre. Por último Caja-9 valor (-3) Caja-10 valor (-2) posicionando Caja-10 encima de Caja-9 y ambas detras del elemento padre.
