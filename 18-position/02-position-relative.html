position: relative;

  • Mantiene su flujo de renderizado.
  • No pierde su espacio reservado.
  • Permite el uso de top, left, etc.
  • Al utilizar las propiedades top, left, etc, se mueven los elementos a partir de su posicion original.

Ejemplos:

Original


.bloque9 {
    background-color: lightsalmon;
    margin-top: 95px;
}

.bloque10{
    background-color: lightseagreen;
    margin-top: 10px;
}

.bloque9, .bloque10{
    width: 50px;
    height: 50px;
    margin-left: auto;
    margin-right: auto;
}
