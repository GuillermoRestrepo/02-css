ESTILOS EXTERNOS

Los estilos externos consisten en escribir el CSS en un archivo separado con extensión .css y luego enlazarlo al documento HTML.

Ejemplo:

Código Resultado

Código HTML:

<head>
    <link rel="stylesheet" href="./03-external-styles.css">
</head>

<body>
    <span class="le1">Elemento de línea #1</span>
</body>
 Elemento de línea #1

Código CSS:

.le1 {
    color: light;
}
