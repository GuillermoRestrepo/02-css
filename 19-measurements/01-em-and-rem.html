em & rem

Son medidas relativas que toman como referencia el tamaño de la fuente de la letra (font-size: ;).

rem

  • Normalmente se utiliza para definir el tamaño de los textos de una página web.
  • Un (rem) equivale al ancho del caracter "M" del elemento raíz (<html>), mismo que por defecto mide 16 píxeles.

Ejemplo de (rem)


.text-1 {
    fot-size: 16px;
}
                

.text-3 {
    fot-size: 32px;
}
                

.text-5 {
    fot-size: 48px;
}
                

.text-7 {
    fot-size: 64px;
}

em

  • Normalmente se utilizado para definir el tamaño de alguna propiedad que necesita ser proporcional al tamaño de la letra del elemento, ya sea q esta aumente o disminuya.
  • Un (em) equivale al ancho del caracter "M" del elemento en donde se encuentra.

NOTAS

  • Se utiliza como referencia la letra mayúscula "M", porque es el caracter más ancho del alfabeto latino.
  • No es aconsejable cambiar el tamaño de la letra del elemento raíz <html>, ya que este sirve como referencia al momento de dar tamaño a otros textos del documento.
  • El tamaño de letra de 16 píxeles que por defecto tiene el elemento raíz, se aproxima al tamaño de lera 12 de documentos.
  • Al ser parecido al tamaño 12, nos resulta familiar, y al cambiar este tamaño nos resultará extraño.
