ESTILOS INTERNOS

Es una forma de agregar CSS dentro del mismo archivo HTML, pero separado del contenido, usando la etiqueta <style> colocada dentro del <head>.

Ejemplo:

Código Resultado

Código HTML

<head>
    <style>
        .le1 {
            color: lightseagreen;
        }
    </style>                                
</head>
<body>
    <span class="le1">Elemento de línea #1.</span>
</body>
 Elemento de línea #1.
