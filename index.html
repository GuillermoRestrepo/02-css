INTRODUCCIÓN A CSS

¿Qué Es CSS?

CSS (Cascading Style Sheets) es un lenguaje que se usa para dar estilo y diseño a las páginas web. Mientras que HTML define la estructura del contenido (por ejemplo, encabezados, párrafos, imágenes), CSS se encarga de cómo se ven esos elementos: colores, tamaños, posiciones, márgenes, fuentes, etc.

¿Cómo Se Usa CSS?

Hay tres formas principales de usar CSS en un documento HTML:

  1. CSS En Línea (inline)

    Se aplica directamente al elemento HTML.

    
                    <p style="color: red;">Este texto es rojo</p>

  2. CSS Interno (internal)

    Se coloca dentro de una etiqueta <style> en el <head> del HTML.

    
                    <head>
                        <style>
                            p {
                                color: blue;
                            }
                        </style>
                    </head>

  3. CSS Externo (external)

    Se escribe en un archivo aparte (por ejemplo, estilos.css) y se enlaza al HTML con .

    Archivo HTML:

    
                    <head>
                        <link rel="stylesheet" href="estilos.css">
                    </head>

    Archivo CSS:

    
                    p {
                        color: green;
                      }

Sintaxis Básica De CSS

  • Selector: Indica a qué elemento HTML se aplican los estilos.
  • Propiedad: Lo que se desea modificar.
  • Valor: El valor que se le asigna a una propiedad.
  • Declaración: Es el conjunto de una propiedad y su valor.
  • Regla: Es el conjunto de selector(es) y declaracion(es), es decir, puede haber más de un selector y mas de una declaracion en una misma regla.

Ejemplo:


            selector1, Selector2, ..., SelectorN {
                propiedad1: valor;
                propiedad2: valor;
                ...
                propiedadN: valor;
              }
