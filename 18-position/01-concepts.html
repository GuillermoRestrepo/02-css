CONCEPTOS DE POSICIONAMIENTO

  • Fujo De Rederizado: es la forma en la que un navegador carga el contenido de una página web, lo hace de arriba hacia abajo y de izquierda a derecha.

  • Punto De Origen De Un Elemento: a partir de este punto se calculará el desplazamiento de un elemento, es la esquina superior izquierda.

  • Espacio Reservado: Es el espacio que ocupa un elemento pintado en el navegador.

  • Elemento Posicionado: se considera que un elemento está posicionado, cuando el valor de la propiedad (position) de este elemento es diferente a (static), siendo este el valor por defecto de todo elemento.

  • Stackin context (Contexto De Apilamiento): es el orden en el que se superponen los elementos dentro de un contenedro.

  • ¿Qué Pasa Caundo Se Posiciona Un Elemento?: se habilitan 5 propiedades que permiten mover el elemento en los 3 ejes, horizontal, vertical y profundidad. Estas propiedades son:

    • top: sirve para mover un elemento desde la parte superior de su contenedor, si el valor es positivo, mueve el elemento hacia abajo, si es negatio lo mueve hacia arriba.

    • right: sirve para mover un elemento desde la parte derecha de su contenedor, si el valor es positivo, se mueve hacia la izquierda, si es negativo, hacia la derecha.

    • bottom: sirve para mover un elemento desde la parte inferior de su contenedor, si el valor es positivo, mueve el elemento hacia arriba, si es negatio lo mueve hacia abajo.

    • left: sirve para mover un elemento desde la parte izquierda de su contenedor, si el valor es positivo, se mueve hacia la derecha, si es negativo, hacia la izquierda.

    • z-index: sirve para indicar su posición en el eje z que representa la profundidad, es decir, cual elemento se superpone a caul.

    • NOTA: Si se utiliza la propiedad top o right, automaticamente inutilizan las propiedades bottom o left respectivamente.

Posibles Valores De La Propiedad Position

  • static (Valor Por Defecto):

    • El elemento se posiciona según el flujo normal del documento.
    • No se puede usar las propiedades top, left, etc.

  • relative

  • absolute

  • fixed

  • sticky
