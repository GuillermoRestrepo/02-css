PROPIEDAD (border-radius)

Es una propiedad que se utiliza para redondear las esquinas de un elemento.

Se puede hacer de forma circular u ovalada.

Borde Circular

  1. Borde circulares iguales en las 4 esquinas:

    Código HTML:

    
<body>
    <div class="eb1"></div>
</body>

    Código CSS:

    
.eb1 {
    background-color: orange;
    width: 250px;
    height: 250px;
    border-radius: 30px;
}

  2. Borde circulares diferentes valores en cada esquina:

    En Varias Líneas En Una Sola Línea

    Código HTML:

    
<body>
    <div class="eb2"></div>
</body>

    Código CSS:

    
.eb2 {
    background-color: orange;
    width: 250px;
    height: 250px;

    border-top-left-radius: 20px;
    border-top-right-radius: 40px;
    border-bottom-right-radius: 60px;
    border-bottom-left-radius: 80px;
}

    Código HTML:

    
<body>
    <div class="eb2"></div>
</body>

    Código CSS:

    
.eb3 {
    background-color: orange;
    width: 250px;
    height: 250px;

    border-radius: 80px 60px 40px 20px;
}

Borde Óvalado

NOTA: (Eje horizontal / Eje vertical)

  1. Bordes óvalados igualmente en las 4 esquinas:

    Código HTML:

    
<body>
    <div class="eb4"></div>
</body>

    Código CSS:

    
.eb4 {
    background-color: orange;
    width: 250px;
    height: 250px;
    border-radius: 40px / 90px;
}

  2. Bordes óvalados con diferentes valores en cada esquina:

    En Varias Líneas En Una Sola Línea

    Código HTML:

    
<body>
    <div class="eb5"></div>
</body>

    Código CSS:

    
.eb5 {
    background-color: orange;
    width: 250px;
    height: 250px;

    border-top-left-radius: 10px 40px;
    border-top-right-radius: 30px 60px;
    border-bottom-right-radius: 50px 80px;
    border-bottom-left-radius: 70px 100px;
}

    Código HTML:

    
<body>
    <div class="eb2"></div>
</body>

    Código CSS:

    
.eb6 {
    background-color: orange;
    width: 250px;
    height: 250px;

    border-radius: 70px 50px 30px 10px / 100px 80px 60px 40px;
}
