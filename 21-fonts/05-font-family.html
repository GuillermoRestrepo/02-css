PROPIEDAD - (font-family)

Esta propiedad sirve para definir la fuente que tendra un texto.

font-family especifica una lista de fuentes que el navegador debe intentar aplicar al texto. Si no puede usar la primera, pasará a la siguiente, y así sucesivamente hasta encontrar una fuente disponible. Ejemplo:


p {
  font-family: "Open Sans", Arial, sans-serif;
}

Esto significa:

  1. Usa "Open Sans" si está disponible.
  2. Si no, usa Arial.
  3. Si tampoco está, usa cualquier fuente sans-serif disponible en el sistema.

Valores Predeterminados

Son varios tipos de fuentes que el navegador siempre puede interpretar:

serif Fuentes con serifas (ej: Times, Georgia)
sans-serif Fuentes sin serifas (ej: Arial, Helvetica)
monospace Todas las letras ocupan el mismo ancho (ej: Courier New)
cursive Parecen escritura a mano
fantasy Decorativas, poco comunes
system-ui Usa la fuente del sistema operativo (muy útil para apps modernas)

Ejemplo


font-family: serif;
                

font-family: sans-serif;
                

font-family: monospace;
                

font-family: cursive;
                

font-family: fantasy;
                

font-family: system-ui;

Notas:

  • Para mejorar la lectura es preferible utilizar fuentes sin serifa.
  • Si el nombre de una fuente contiene espacios, hay que ponerlo entre comillas.
  • Se puede utilizar diferentes tipos de fuentes provenientes de diferentes recursos, como (google fonts).
  • De la misma manera, no es aconsejable utilizar mas de 2 tipos de fuente en un sitio/página web.
