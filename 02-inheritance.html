HERENCIA

La herencia es cuando un elemento hijo toma automáticamente ciertos estilos de su elemento padre. Ejemplo:

Código HTML


<body>
    <h3 class="encabezado">COMO FUNCIONA LA <span>HERENCIA</span> EN CSS</h1>
</body>

Código CSS


.encabezado {
    color: cyan;
}

COMO FUNCIONA LA HERENCIA EN CSS

En este ejemplo se puede ver como la etiqueta (<span>) hereda la propiedad (color) de la etiqueta que lo contiene (etiqueta padre), que viene siendo (<h3>).

Existen propiedades que se heredan como (color), y propiedades que no lo hacen, como (margin). Ejemplo:

Código HTML


<body>
    <h3 class="encabezado-2">COMO FUNCIONA LA <span class="editar">HERENCIA</span> EN CSS</h1>
</body>

Código CSS


.encabezado-2 {
    margin: 20px;
}

Existen elementos que son capaces de heredar estilos, como (<span>) y otros que no lo son como (<a>).

Código HTML


<body>
    <h3 class="encabezado">Esto es un <span>span</span>.</h3>
    <h3 class="encabezado">Esto es un <a href="#">enlace</a>.</h3>
</body>

Código CSS


.encabezado {
    color: cyan;
}

Esto es un span.

Esto es un enlace.

Se puede ver como la etiqueta (<span>) hereda los estilos, mientras que la etiqueta (<a>).

Código HTML


<body>
    <h3 class="encabezado">Esto es un <span class="span">span</span>.</h3>
    <h3 class="encabezado">Esto es un <a class="enlace" href="#">enlace</a>.</h3>
</body>

Código CSS


.encabezado {
    color: cyan;
}

.span {
    color: initial;
}

.enlace {
    color: inherit;
}

Esto es un span.

Esto es un enlace.

initial: sirve para que un elemento se fuerce a NO heredar estilos.

inherit: sirve para que un elemento se fuerce a SI heredar estilos.
