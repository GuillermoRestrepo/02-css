TRUCO

Muchas veces el elemento (body) no ocupa el 100% del viewpor, cuando se desea tener un pie de pagina que vaya pegado a la parte inferior del elemento (body), queda flotando ya que se pega al (body) pero no al viewport, es aqui cuando se usa un (min-height: 100vh;) para que el body ocupe todo el tamaño del viewport.

(min-width: ;)

Sirve para indicar cual va a ser el ANCHO MÍNIMO que puede tener un elemento.

Ejemplo:


.box-1 {
    min-width: 400px;
}

(max-width: ;)

Sirve para indicar cual va a ser el ANCHO MÁXIMO que puede tener un elemento.

Ejemplo:


.box-2 {
    max-width: 500px;
}

(max-height: ;)

Sirve para indicar cual va a ser el ALTO MÁXIMO que puede tener un elemento.

Ejemplo:


.box-4 {
    max-height: 200px;
}

(min-height: ;)

Sirve para indicar cual va a ser el ALTO MÍNIMO que puede tener un elemento.

Ejemplo:


.box-3 {
    min-height: 200px;
}
