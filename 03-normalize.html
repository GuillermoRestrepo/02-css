Normalizar Estilos

  • Por defecto cada navegador tiene ciertos estilos aplicados.
  • Es posible que no todos los navegadores tengan los mismos estilos aplicados por defecto.
  • También es posible que esto cause problemas al visualizar un sitio web desde diferentes navegadores.
  • Para que una página web q "no tenga" estilos propios, se vea de la misma forma en los diferentes navegadores, hay q normalizarla.
  • Para normalizarla hay que relacionar un archivo CSS a nuestra página web, el cual contenga estilos simples, q puedan ser visualizados en cualquer navegador, o en su defecto en la mayoría de estos.
  • Uno de los estilos normalizados mas famosos solo los estilos de Normalize.css

  • Simplemente se descarga el archivo css, se lo coloca dentro del proyecto, y se lo pone dentro de <head> antes de los estilos que se le dará a nuestro sitio web. Ejemplo:

    
                <head>
                    <link rel="stylesheet" href="normalize.css">
                    <link rel="stylesheet" href="estilos-propios.css">
                </head>
