position: sticky;

  • Funciona como (position: fixed), pero dentro de un contenedor especifico.
  • Las propiedades (top), (right), indica q tan cerca tiene q estar el borde del contenedor q se indica, para que este se desplace o se detenga.
  • NOTA: el contenedor del elemento posicionado, tiene que tener el valor (visible) en la propiedad (overflow), es el valor por defecto, en tal caso no usar dicha propiedad.

Ejemplos:

Como el "menú de navegación" se encuenta contenido en el "body", su desplazamiento comprende todo el "viewport" Como los titulos de los articulos se encuentran contenidos en los articulos, el desplazamiento de los titulos solo comprende el articulo q le corresponde. 

.body {
    background-color: #333;
    color: #fff;
    width: 300px;
    height: 300px;
    margin-left: auto;
    margin-right: auto;
    padding: 0px;
    overflow: auto;
}

.main {
    width: 100%;
    height: 100%;
    margin: 0px;
    padding: 10px;
}

.header {
    background-color: lightseagreen;
    text-align: center;
    width: 100%;
    height: 30px;
    padding-top: 7px;
    position: sticky;
    top: 0px;
}
