ESPECIFICIDAD

La especificidad en CSS es el nivel de importancia que tiene un selector para aplicar un estilo cuando hay conflictos entre varios estilos.

¿Cómo calcular la especificidad en CSS?

La especificidad se calcula como cuatro números separados por comas: (A, B, C, D)

Cada letra representa:

Letra Selector
A Estilos en línea (style= "...")
B Identificadores (#id)
C Clases (.class), Atributos ([src= "..."]) y pseudoclases (:hover)
D Etiquetas (<h1>) y pseudo-elementos (::before)

Ejemplo:

Selector Especificidad
h1 (0, 0, 0, 1)
.boton (0, 0, 1, 0)
#menu (0, 1, 0, 0)
p.intro (0, 0, 1, 1)
#header .nav li (0, 1, 1, 1)
style="color: red" (1, 0, 0, 0)

En resumen, la regla que tenga el selector con mayor numero de especificidad, será la regla que defina el estilo.

Por otro lado, si dos reglas tienen el mismo nivel de especificidad, se aplicarán los estilos de la última regla definida.

Ejemplo En Código:

En HTML:


        <body>
            <p id="p1" class="parrafo">Este es un párrafo.</p>
        <body>

En CSS:


        #p1 {
            backgroud-color: yellow;
        }

        .parrafo {
            backgroud-color: blue;
        }

        p {
            backgroud-color: red;
        }

A pesar de que la regla con el selectopr (p) fue la última en ser definida, la regla q validará el navegador será la que tiene el selector (#p1) y en caso de q se borre dicha regla, la segunda con mayor especificidad seria la regla que tiene el selector (.parrafo).

Nota: utilizar !important para definir una propiedad de una regla como la maás importante, es una muy mala práctica, es recomendable no acudir a su uso.

Nota #2: lo mas recomendable es utilizar un unico tipo de selectoy y así evitarse problemas.
