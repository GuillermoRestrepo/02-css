REGLA IMPORT

La regla @import se utiliza para importar una hoja de estilos CSS dentro de código CSS.

Ejemplo:

Código Resultado

Código HTML:

<head>
    <style>
        @import url(../css/04-import-rule.css);
    </style>
</head>

<body>
    <span class="le1">Elemento de línea #1</span>
</body>
 Elemento de línea #1

Código CSS:

.le1 {
    color: lightseagreen;
}
