NOTA: PARA VER EL FUNCIONAMIENTO HAY Q CAMBIAR EL TAMAÑO DE LA VENTANA DEL NAVEGADOR

(vw)

Cálcula el tamaño del elemento, tomando como referencia el ancho del (viewport) del navegador.

Ejemplo:


.box-1 {
    width: 30vw;
}

(vh)

Cálcula el tamaño del elemento, tomando como referencia el alto del (viewport) del navegador

Ejemplo:


.box-2 {
    width: 40vh;
}

(vmin)

Cálcula el tamaño del elemento, tomando como referencia el valor más pequeño entre el alto y el ancho del (viewport)

Ejemplo:


.box-3 {
    width: 30vmin;
}

(vmax)

Cálcula el tamaño del elemento, tomando como referencia el valor más grande entre el alto y el ancho del (viewport)

Ejemplo:


.box-4 {
    width: 40vmax;
}

NOTA: cuando se use vmin y vmax, todo dependerá de las dimensiones del viewpor y si este cambia de tamaño.
