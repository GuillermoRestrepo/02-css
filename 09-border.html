PROPIEDAD border

Como su nombre lo indica, sirve para definir el borde de un elemento. (modifica el tamaño del elemento)

Por defecto, la propiedad border cuenta con 3 valores:


        /*
            border: width style color;
        */
        border: 1px solid black;

Puede dividirse por su estilo


        border-width: 1px;
        border-style: solid;
        border-color: black;

También puede dividirse por lado:


        /*  Borde superior.  */
        border-top: 1px solid yellow;
        
        /*  Borde derecho.  */
        border-right: 3px solid blue;
        
        /*  Borde inferior.  */
        border-bottom: 5px solid red;
        
        /*  Borde izquierdo.  */
        border-left: 7px solid green;

También puede subdividirse en ambas a la vez:


        border-top-width: 1px;
        border-right-style: solid;
        border-bottom-color: black;
        border-left-width: 3px;

Posibles Valores Para border-width:

Valores númericos:

  • px: píxeles, son valores fijos.
  • em, rem: Relativos al tamaño de la fuente.

palabras clave:

  • thin: Borde delgado (1px aproximadamente)
  • medium: Borde medio (3px aproximadamente)
  • thick: Borde grueso (5px aproximadamente)

Posibles Valores Para border-style:

Estilos 2D:

  • none (sin borde)

  • solid (línea sólida)

  • dashed (línea discontinua)

  • dotted (punteado)

  • double (doble línea)
